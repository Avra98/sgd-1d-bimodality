# 1-D SGD · iterate shape beyond stochastic stability

Interactive browser lab for the scalar cocycle with random curvature
(Gaussian / Uniform / Bernoulli / χ²), knobs for η, variance, α, β,
additive noise, and seed, plus analytical edges η_MS, η_L, η̂_H.

**Live demo:** https://avra98.github.io/sgd-1d-bimodality/
